## How This Works

During session learning capture, Claude watches for repeated workflows. These get logged in session learnings tagged `[workflow]`. During weekly retros, if the same workflow appears 2+ times, it gets promoted here. Claude proposes the automation type and implementation. You approve, then it gets built.
